# PROGRAMAÇÃO MULTITHREAD

Esse projeto faz parte da Atividade Prática 04


Clone o projeto, e no diretório raiz, execute em uma linha de comando:

```
make procon
Compila o arquivo src/prodCons.c do problema Produtor/Consumidor e salva na pas build
./build/procon
Executa o arquivo após compilação

make planta
Compila o arquivo src/plantacao.c  do problema Plantação e salva na pas build
./build/planta
Executa o arquivo após compilação

make dorminhoco
Compila o arquivo src/monitorDorminhoco.c do problema Monitor Dorminhoco e salva na pas build
./build/monitorDorminhoco
Executa o arquivo após compilação

make barbearia
Compila o arquivo src/Barbearia/barbearia.c do problema Barbearia e salva na pas build
./build/barbearia
Executa o arquivo após compilação

make plantacao2
Compila o arquivo src/Outros/plantacao2.c do problema Plantação (outra versão) e salva na pas build
./build/plantacao2
Executa o arquivo após compilação

make prodConBuf
Compila o arquivo src/Outros/prodConBuf.c do problema Produtor/Consumidor (outra versão) e salva na pas build
./build/prodConBuf
Executa o arquivo após compilação
```
